git touch - создать файл
git add <имя файла>- подготовить к добавлению
git status - 
git log - лог операций
cat <имя файла> - отобразить содержимое файла в консоль
git commit -m 'Текст сообщения добавляемое в коммит'
git push - тправить в удаленный репозиторий

В первый раз эту команду нужно вызвать с флагом -u и параметрами origin (имя удалённого репозитория) и main или master (название текущей ветки). 
Флаг -u свяжет локальную ветку с одноимённой удалённой. 
git push -u origin master
